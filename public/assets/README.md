# Assets do site Guruva

Coloque o vídeo cinematográfico do hero neste diretório com o nome:

```text
guruva-hero.mp4
```

Direção sugerida: drone baixo sobre os vinhedos às 6h da manhã, trabalhadores colhendo uvas, seleção no packing house, câmara fria, contêiner sendo lacrado e navio saindo do porto.

Enquanto o arquivo não existir, o navegador usa a imagem de poster definida no HTML.
